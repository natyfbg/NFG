Recommended Workflow
Build and ship a structured program
  1. Create or clean workouts in the library, then run Workout Quality and metadata backfill.
  2. Create a hub program for the public entry point on /programs.
  3. Build tracks for level and environment variants from the Programs screen.
  4. Scaffold weeks inside each track with a default split.
  5. Add week items and link exercises for each day in the split.
  6. Run Link Health and readiness checks before switching status.
  7. Publish + Activate + Show on Home when the content is ready for members.
Production Areas
Main content systems
Workouts Exercise library, media, tips, and metadata quality. Programs Hub cards, tracks, readiness, publish state, and Home visibility. Program Weeks Week containers and scaffold tools for each track. Link Health / Readiness Broken workout links, unresolved items, and publish blockers. Styles Controlled workout style labels used by the library. Home Plans Legacy/supporting Home content. Programs are the current primary Home strategy.
